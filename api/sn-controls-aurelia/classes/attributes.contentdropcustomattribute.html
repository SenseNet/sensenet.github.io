Class ContentDropCustomAttribute

Custom attribute that allows you to handle custom content, contentlist or file drop events on your HTML element. Usage example:

<div class="my-droptarget-element"
     content-drop.call="dropContent(event, stringifiedContent, stringifiedContentList, files)"></div>

The defined dropContent(..args) method will be called on Drop events. This attribute will stop the event propagation for drag & drop events, so it can be nested.
