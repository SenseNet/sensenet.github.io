A flexible development platform

A central repository with an extendable API ready for integration

  • Modern web technologies

    Do not worry about the backend! Use the client side technology of your choice (for example React or Aurelia with Redux) to build your UI and business logic on top of our Content Repository REST API using the sensenet ECM JavaScript SDK.

  • Flexible back-end

    Use or extend our wide server-side .Net API, integrate sensenet ECM into your MVC web application, or even replace one of our pluggable components with your own custom provider!

  • Growing widget stock

    You are not alone! Our community builds plugins for the many extensibility points of sensenet ECM. Use them or build your own components for the benefit of your clients or the community.
